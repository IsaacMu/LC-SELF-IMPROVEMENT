#203.Â Remove Linked List Elements

------

Remove all elements from a linked list of integers that have value **val**.

**Example:**

```
Input:  1->2->6->3->4->5->6, val = 6
Output: 1->2->3->4->5
```

# Solution

```java
 */
class Solution {
    public ListNode removeElements(ListNode head, int val) {
        ListNode p,pre,re;
        pre = new ListNode(0);
        re = pre;
        pre.next = head;
        p = head;
        while(p != null){
            if(p.val == val) {
                pre.next = p.next;
            }else pre = p;
            p = p.next;
        }
        return re.next;
    }
}
```

Time is O(n) Space is O(1)
