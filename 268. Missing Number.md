# Solution

```java
class Solution {
    public int missingNumber(int[] nums) {
        Set<Integer> set = new HashSet<Integer>();
        for (int i = 0; i < nums.length + 1; i++) {
            set.add(i);
        }
        for(Integer t: nums){
            set.remove(t);
        }
        for(Integer t: set){
            return t;
        }
        return 0;
    }
}
```
