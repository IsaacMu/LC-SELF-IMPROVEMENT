# 1004. Max Consecutive Ones III
 

```python
class Solution:
    def longestOnes(self, nums: List[int], k: int) -> int:
        l = 0
        for r in range(len(nums)):
            k -= 1-nums[r]
            if k < 0:
                k += 1- nums[l]
                l += 1
        return r - l + 1
```

# Notes

Time Complexity is O(n), space is O(1)
