# 32. Longest Valid Parentheses

#  Solution

```python
class Solution:
    def longestValidParentheses(self, s: str) -> int:
        n = len(s)
        dp = [0 for _ in range(n)]
        res = 0
        for i in range(1, n):
            if s[i] == ')':
                if s[i-1] == '(':
                    if i > 1:
                        dp[i] = dp[i-2] + 2
                    else:
                        dp[i] = 2
                elif i - dp[i-1] > 0 and s[i-dp[i-1]-1] == '(':
                    dp[i] = dp[i-1] + 2
                    if i - dp[i-1] > 1:
                        dp[i] += dp[i-dp[i-1]-2]
                res = max(res, dp[i])
        return res
```

# NOTE:

DP, O(N) for both time and space.
