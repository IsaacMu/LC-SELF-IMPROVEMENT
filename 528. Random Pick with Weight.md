# 528. Random Pick with Weight

```python
class Solution:

    def __init__(self, w: List[int]):
        self.w = w
        self.c = [0] * len(w)
        s = sum(w)
        self.c[0] = w[0]/s
        for i in range(1, len(w)):
            self.c[i] = self.c[i-1] + w[i]/s

    def pickIndex(self) -> int:
        pr = random.random()
        return bisect.bisect_left(self.c, pr)
```


O(log N), O(n)
