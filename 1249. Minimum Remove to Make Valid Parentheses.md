# 1249. Minimum Remove to Make Valid Parentheses

```python

class Solution:
    def minRemoveToMakeValid(self, s: str) -> str:
        stack = []
        for i in range(len(s)):
            ch = s[i]
            if ch == '(':
                stack.append(('(', i))
            if ch == ')':
                if stack and stack[-1][0] == '(':
                    stack.pop()
                else:
                    stack.append((')', i))
        res = ""
        start = 0
        for ch, index in stack:
            res += s[start:index]
            start = index+1
        res += s[start:]
        return res
```

# Notes

1. Time Complexity is O(n), space Complexity is O(n)
